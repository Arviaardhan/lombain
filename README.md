# lombain - Frontend 🚀

**lombain** adalah platform kolaborasi mahasiswa untuk menemukan rekan tim terbaik guna menghadapi kompetisi/lomba. Dibangun dengan fokus pada efisiensi manajemen tim dan kecocokan skill anggota.

## 🛠 Tech Stack
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn/UI
- **Animation**: Framer Motion & GSAP
- **State Management**: React Hooks
- **Icons**: Lucide React
- **HTTP Client**: Fetch API

## 🚀 Fitur Utama
- **Talent Discovery**: Mencari rekan tim berdasarkan skill, jurusan, dan portofolio.
- **Advanced Team Management**: Kelola struktur tim dengan sistem drag-and-drop.
- **Waiting List System**: Fitur antrean pendaftar sebelum dimasukkan ke formasi utama tim.
- **Real-time UI Update**: Sinkronisasi status tim yang instan menggunakan Optimistic UI.
- **Team Locking Mechanism**: Fitur untuk mengunci formasi tim jika sudah final.
